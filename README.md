# 📰 NEW NEWS

![Banner do projeto](imagens/banner.jpg)

## 📋 Sobre o Projeto

Este é o site da agência de jornalismo [NEW NEWS], desenvolvido como parte do Projeto Integrador do Curso Técnico em Desenvolvimento de Sistemas.

O site contém notícias sobre o cotidiano escolar, além de conteúdo multimídia como vídeos e podcasts produzidos via canva.

## 🚀 Como acessar

O site está publicado no GitHub Pages e pode ser acessado pelo link:
🔗 [https://karen-reis-dev.github.io/NEW-NEWS-PROJ](https://karen-reis-dev.github.io/NEW-NEWS-PROJ)

## 🛠️ Tecnologias utilizadas

- **HTML5** - Estrutura das páginas
- **CSS3** - Estilização e layout responsivo
- **JavaScript** - Interatividade e funcionalidades
- **GitHub Pages** - Hospedagem gratuita
